## Italian Brainrot Merge Fella Game

A simple terminal game where you merge fellas on a 4x4 grid. Use the `W`, `A`, `S`, `D` keys to move the tiles. Two tiles with the same number will merge into a higher-level fella. Press `Q` to quit.

Run the game with:

```bash
python italian_brainrot_merge_fella.py
```
